## Python 
### Installieren
1. Python installieren: https://www.python.org/
2. Entwicklungsumgebung erstellen
3. Bibliotheken installieren (in Entwicklungsumgebung): ```pip install -r code/requirements.txt``` 

### Ausführen
```python app.py```
